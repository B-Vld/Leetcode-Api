Leetcode Api
GitHub
Comes with a discord webhook integration scheduled to post the daily challenge in a desired text channel
Ways to structure a webhook : https://birdie0.github.io/discord-webhooks-guide/discord_webhook.html

Example calls :

1. Fetching the daily challenge

https://api4leetcode.herokuapp.com/api/daily-challenge

2. Fetching a random question based on difficulty levels. Specify the difficulty EASY / MEDIUM / HARD in the {difficulty} block

https://api4leetcode.herokuapp.com/api/daily-challenge/{difficulty}
